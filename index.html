Jogo de Ping Pong

Este jogo de Ping-Pong é desenvolvido com a biblioteca p5.js e simula o clássico jogo de tênis de mesa. Ele consiste em uma bolinha que se move pela tela e duas raquetes: uma controlada pelo jogador e outra pelo bot. O objetivo é evitar que a bolinha ultrapasse a raquete controlada pelo jogador, ao mesmo tempo tentando fazer a bolinha passar pela raquete do bot para marcar pontos. O bot se movimenta automaticamente, ajustando sua posição conforme a bolinha. O jogo continua até que um dos jogadores atinja um número pré-definido de pontos. A implementação envolve detecção de colisões, movimentação das raquetes e controle de pontuação.

Interface de um aplicativo

"Este protótipo de aplicativo foi desenvolvido no Figma, uma ferramenta amplamente utilizada para design de interfaces e protótipos interativos, como parte de um processo de aprendizado baseado nas aulas oferecidas pela plataforma Alura. O curso forneceu uma base sólida para entender as melhores práticas de design de interfaces e como aplicar essas técnicas no Figma, criando layouts funcionais e visualmente agradáveis. O protótipo visa resolver um problema específico, oferecendo uma experiência de usuário fluida e intuitiva. A partir dos conceitos aprendidos nas aulas, foram implementadas interações e fluxos que proporcionam uma navegação eficiente e clara, tornando o aplicativo prático e fácil de usar."

Catálogo de filmes

"Este catálogo de filmes e vídeos funcional foi criado utilizando o GitHub para controle de versão e o p5.js para desenvolvimento interativo, com base nos ensinamentos das aulas da plataforma Alura. A partir do curso, foram adquiridos conhecimentos essenciais para implementar uma aplicação web que organiza e exibe informações sobre filmes e vídeos de maneira intuitiva e dinâmica. O p5.js foi utilizado para criar elementos visuais interativos, enquanto o GitHub foi fundamental para o versionamento do código e para o gerenciamento do projeto. O sistema permite ao usuário explorar diferentes filmes e vídeos, com funcionalidades de busca e categorização, proporcionando uma experiência rica e eficiente."
